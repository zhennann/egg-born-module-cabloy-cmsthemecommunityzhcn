# egg-born-module-cabloy-cmsthemecommunityzhcn

Cabloy-CMS Theme

## Installation

``` bash
$ npm i egg-born-module-cabloy-cmsthemecommunityzhcn
```

## Usage

Changing languange's theme to `cabloy-cmsthemecommunityzhcn`

``` javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cabloy-cmsthemecommunityzhcn"
}
```
